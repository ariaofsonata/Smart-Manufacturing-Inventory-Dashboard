\# 📊 智慧製造庫存監控儀表板

這是一個結合 \*\*Python + MySQL + Streamlit\*\* 的端到端數據分析專案。



\## 🌟 核心功能

\* \*\*即時監控\*\*：直接串接 MySQL 資料庫顯示最新庫存。

\* \*\*交互式篩選\*\*：透過滑桿即時過濾產品價格範圍。

\* \*\*自動化統計\*\*：自動計算產品總數與平均單價。



\## 🛠️ 技術棧

\* \*\*語言\*\*: Python

\* \*\*資料庫\*\*: MySQL

\* \*\*視覺化\*\*: Streamlit, Pandas



\## 🚀 快速啟動

1\. 安裝環境：`pip install -r requirements.txt`

2\. 初始化資料庫：`python init\_db.py`

3\. 匯入範例數據：`python insert\_data.py`

4\. 啟動網頁：`streamlit run app.py`



> \*\*注意\*\*：請在程式碼中將 `password` 修改為您的 MySQL 密碼。
